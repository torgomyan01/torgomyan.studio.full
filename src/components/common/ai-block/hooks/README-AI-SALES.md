# AI Sales Engine - Վաճառքային AI համակարգ

## Նկարագրություն

Սա մինի AI համակարգ է, որը նախատեսված է chat bot-ում վաճառքները բարելավելու և օգտատերերին համոզելու համար:

## Հիմնական հնարավորություններ

### 1. **Պատասխանների վերլուծություն** (`analyzeUserAnswer`)

- Վերլուծում է օգտատիրոջ պատասխանները և որոշում նրանց տրամադրությունը
- Որոշում է օգտատիրոջ մտադրությունները (հետաքրքրված, տատանվող, առարկող, պատրաստ գնելու)
- Բացահայտում է հիմնական բառերը և բանալի բառերը
- Որոշում է, թե արդյոք անհրաժեշտ է upsell կամ համոզում

### 2. **Համոզման ռազմավարություններ** (`generatePersuasionStrategy`)

Համակարգը կարող է հասկանալ և պատասխանել տարբեր տեսակի առարկություններին:

- **Բյուջեի առարկություններ** ("дорого", "бюджет")
  - Ցույց է տալիս ROI և երկարաժամկետ օգուտներ
  - Ներկայացնում է վիճակագրություններ
  - Ստեղծում է անհետաձգելիության զգացում

- **Անորոշություն** ("не нужно", "не уверен", "пока нет")
  - Ներկայացնում է օգուտներ և առավելություններ
  - Օգտագործում է սոցիալական ապացույցներ
  - Համեմատում է մրցակիցների հետ

- **Բարդություն** ("слишком сложно", "не разберусь")
  - Վստահություն է տալիս աջակցության մասին
  - Ներկայացնում է պարզության երաշխիքներ
  - Օգտագործում է հաջողության պատմություններ

- **Ժամանակի բացակայություն** ("времени нет", "некогда")
  - Ցույց է տալիս, թե ինչպես է ավտոմատացումը խնայում ժամանակ
  - Ներկայացնում է ավտոմատացման օգուտները

- **Մրցակցություն** ("конкуренты", "уже есть")
  - Ցույց է տալիս մրցակցային առավելությունները
  - Ստեղծում է անհետաձգելիության զգացում

### 3. **Upsell առաջարկներ** (`generateUpsellProposal`)

- Վերլուծում է օգտատիրոջ պատասխանները և բացահայտում լրացուցիչ ծառայությունների անհրաժեշտություն
- Առաջարկում է համապատասխան լրացուցիչ ծառայություններ
- Ներկայացնում է օգուտներ և վիճակագրություններ

### 4. **Ադապտիվ պատասխաններ** (`generateAdaptiveResponse`)

- Ստեղծում է հարմարեցված պատասխաններ օգտատիրոջ տրամադրությանը
- Օգտագործում է համոզման ռազմավարություններ, երբ անհրաժեշտ է
- Առաջարկում է upsell, երբ դա տեղին է
- Ապահովում է բնական և մարդկային խոսակցություն

## Օգտագործում

```typescript
import { useAISalesEngine } from './use-ai-sales-engine';

const { analyzeUserAnswer, generateAdaptiveResponse } = useAISalesEngine();

// Վերլուծել պատասխանը
const analysis = analyzeUserAnswer('Нет, это слишком дорого');

// Ստեղծել ադապտիվ պատասխան
const response = generateAdaptiveResponse(
  question,
  userAnswer,
  service,
  chatData
);
```

## Ինտեգրում

AI համակարգը ինտեգրված է `use-chat-logic.ts`-ում և ավտոմատ կերպով:

- Վերլուծում է յուրաքանչյուր օգտատիրոջ պատասխան
- Գեներացնում է համոզիչ պատասխաններ, երբ անհրաժեշտ է
- Առաջարկում է լրացուցիչ ծառայություններ

## Վաճառքային տեխնիկաներ

1. **Վիճակագրություններ** - Օգտագործում է իրական վիճակագրական տվյալներ
2. **Սոցիալական ապացույցներ** - Հիշատակում է այլ հաճախորդների հաջողությունները
3. **Անհետաձգելիություն** - Ստեղծում է անհետաձգելիության զգացում
4. **Արժեքի առաջարկ** - Ներկայացնում է օգուտներ և ROI
5. **Համեմատություն** - Համեմատում է մրցակիցների հետ

## Ապագա բարելավումներ

- Ավելի խոր վերլուծություն օգտագործելով NLP
- Մեքենայական ուսուցում օգտատիրոջ վարքագծի կանխատեսման համար
- Անձնավորված առաջարկներ օգտատիրոջ պատմության հիման վրա
- A/B թեստավորում տարբեր համոզման ռազմավարությունների համար
